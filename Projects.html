RDR2 Perfection Tracker: View on GitHub

HTML, CSS, JavaScript

I made this tracker to learn javascript and figure out how it works with HTML and CSS. It is an interactive checklist that helps players to stay organized, and saves their selections using local storage. It loads data from a JSON file that is grouped into categories. It also has progress bars that track progress by percentage for both category and overall completion. I wanted to make something meaningful, and this is something that I will actually use when I play!
